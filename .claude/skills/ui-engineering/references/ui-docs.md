# UI Engineering References

## Component Libraries

### Shadcn UI
- Documentation: https://ui.shadcn.com/
- Use for: Forms, modals, dropdowns, primitives
- Installation: `npx shadcn@latest add [component]`

### Aceternity UI
- Documentation: https://ui.aceternity.com/
- Use for: Animations, effects, hero sections
- When to use: Marketing pages, landing sections, visual impact

## Local References

- `components/ui/` - Shadcn primitives
- `components/effects/` - Aceternity components
- `lib/utils/cn.ts` - Class name utility
